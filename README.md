# 🍕 Foodie — Food Delivery & Restaurant Ordering App

A full-featured food delivery and restaurant ordering platform built with Java and MySQL.

---

## 📌 About the Project

Foodie is a Major Project application that allows users to:
- Browse nearby restaurants
- Search and explore food items
- Place food orders online
- Track order status in real time

---

## 🛠️ Tech Stack

| Layer       | Technology      |
|-------------|-----------------|
| Backend     | Java (Core / Servlets / JSP) |
| Database    | MySQL           |
| Server      | Apache Tomcat (Servers folder) |
| Frontend    | HTML, CSS, JavaScript |

---

## ⚙️ How to Run

1. **Clone the repository**
   ```bash
   git clone https://github.com/ankit7847/foodie.git
   ```

2. **Import into Eclipse**
   - Open Eclipse → File → Import → Existing Projects into Workspace
   - Select the cloned folder

3. **Setup MySQL Database**
   - Create a database named `foodie`
   - Import the SQL file from the project (if provided)
   - Update DB credentials in the config/connection file

4. **Run on Server**
   - Right-click project → Run As → Run on Server
   - Select Apache Tomcat
   - Open browser at `http://localhost:8080/foodie`
